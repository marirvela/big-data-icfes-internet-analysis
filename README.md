# Disconnected & Left Behind  
### A Big Data Analysis of Internet Access and Educational Outcomes in Colombia's Caribbean Region

---

This project is developed as part of the Big Data Processing course at Pontificia Universidad Javeriana.

It aims to analyze the relationship between internet access and educational outcomes (ICFES Saber 11 results) across municipalities in Colombia’s Caribbean region using Big Data tools and the CRISP-DM methodology.

---

## Description

Big Data project uncovering how internet access shapes ICFES performance in Colombia’s Caribbean region, using Apache Spark and CRISP-DM to drive data-driven strategies for reducing educational inequality.

---

## Problem Statement

Colombia's Caribbean region presents significant socioeconomic disparities, particularly in rural municipalities where limited internet access may negatively impact educational performance.

The Ministry of Education requires a data-driven approach to understand:
- How internet access affects ICFES results  
- Which municipalities are most affected  
- What actions can improve educational outcomes  

---

## Objective

To identify and analyze the link between internet connectivity and academic performance at the municipal level, in order to propose actionable strategies that improve education quality through digital inclusion.

---

The study focuses on the Colombian Caribbean region, including:

- Atlántico (Barranquilla)  
- Bolívar (Cartagena)  
- Córdoba (Montería)  
- Sucre (Sincelejo)  
- Cesar (Valledupar)  
- Magdalena (Santa Marta)  
- La Guajira (Riohacha)  

---

This project follows the CRISP-DM methodology:

1. Business Understanding  
2. Data Understanding  
3. Data Preparation  
4. Modeling  
5. Evaluation
